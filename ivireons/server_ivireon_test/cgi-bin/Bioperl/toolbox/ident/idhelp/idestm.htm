Estimating Models

Estimate pop-up menu in the ident window provides methods for estimating models based on the Working Data set. The models are inserted into the rectangular placeholders in the Model Board, located in the ident window.

The pop-up has the following commands:

Linear parametric models
Opens a dialog that allows you to generate dynamic linear models with different structures, orders, and delays. A model structure characterizes the relationship between the input and output data, and between unknown noise sources and the output data. Supported model structures include ARX, ARMAX, Output Error (OE), Box-Jenkins (BJ), State-Space, and others.
Process models
Opens a dialog that allows you to generate simple, continuous-time, dynamic linear models - characterized by static gain, time constants, and time delays
Nonlinear models
Opens a dialog that allows you to generate nonlinear dynamic models of two types: Nonlinear ARX models and Hammerstein-Wiener models.
Spectral model
Estimates the system's frequency response from the data by using either Fourier Transform techniques or the Blackman-Tukey approach
Correlation model
Estimates the system's transient response (impulse response) by correlating filtered versions of the input-output data
Quick start
Performs correlation analysis and spectral analysis. Computes a default ARX model with a delay that is determined heuristically based on the estimated impulse response of the system, as well as a default order state-space model.
