Unit Function for Custom Network

The dialog for specifying the unit function for a custom network is launched by:
  • Pressing the Specify... button on the panel for properties of Custom Network under the Model Properties tab of Nonlinear ARX model structure (in Nonlinear Models window).
  • Pressing the Unit Function ... button in the fourth column of the table in the Hammerstein-Wiener model's I/O Nonlinearity panel in the Nonlinear Models window.

The unit function H may be specified in two ways: as a function handle, or as a name of a MATLAB or MEX-file. H should represent a function of the form [f, g, a] = function_name(x), where f is the value of the function, g = df/dx and a indicates the unit function's active range (that is, the derivative g is significantly non zero in the interval [-a a]). This function must be "vectorized": for a vector or matrix x, the output arguments f and g must have the same size as x, computed element-by-element.

Note that it is necessary to specify the unit function when choosing the Custom Network as a form of nonlinearity. The estimation will not work unless a valid unit function has been specified. See the function "gaussunit.m" (type "edit gaussunit" in MATLAB Command Window) as an example of a valid unit function.

How to specify the unit function:
  • Choose the "Function handle" radio button to specify the name or expression for a function handle. The specified variable or expression will be evaluated in the MATLAB base workspace.
  • Choose the "MATLAB or MEX-file" radio button to specify the name of an M file or an MEX file. You can either type in the name of the file (with full path, if the file is not on MATLAB path), or browse for it by clicking on the Browse... button.

After specifying the unit function as a function handle or name of a file, click on Apply or OK button to insert the chosen unit function into the custom network. Hit Cancel to close the dialog.

Refer to the documentation on customnet or type "help customnet" or "idprops customnet" for more information.
