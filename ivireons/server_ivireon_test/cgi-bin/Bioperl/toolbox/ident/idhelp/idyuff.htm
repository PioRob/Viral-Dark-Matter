Frequency Function Data

The Frequency Function view is opened by selecting the corresponding check box in the ident window.

The plots show the frequency responses of selected data sets. For frequency function data sets (IDFRD) (icon has a light-brown background color), the plot gives the amplitude and phase plots of the corresponding frequency response. For time and frequency domain data sets, the plot shows the Empirical Transfer Function Estimate (ETFE).

You set the scales and units in the Style menu.
