Contains commands to manage your sessions. A session consists of the data sets and models, including their diary and notes. Only one session can be open at any time.

Open
Prompts for a filename of a previously saved session.
Merge session
Prompts for a filename of a previously saved session to be merged into the current session.
Close session
Closes the current session. Deletes all data sets and models, empties the Trash, and closes all plot views.
Save session
Saves the current session.
Save as session
Prompts for a filename before saving the current session.
1 - 4 session names:
Four most recent sessions. Selected session is opened and merged directly. The menu initially contains ident example sessions.
Exit ident
Closes all windows associated with ident.
