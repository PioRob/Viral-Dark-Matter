Profiler for Improving Performance

One way to improve the performance of your MATLAB files in pmode and on the cluster is using the parallel profiling tool. The Parallel Computing Toolbox provides the MATLAB file MPI Profiler, a graphical user interface that is based on the results returned by the mpiprofile function.

Use the parallel profiler to help you determine where you can modify your code to make performance improvements.

For details on how to use the MPI Profiler, see the parallel profiler documentation in the Parallel Computing Toolbox.
