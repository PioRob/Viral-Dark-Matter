Toolbox について

Toolbox は、特定の分野の問題を解決するために作られた専門的な M ファイル (MATLAB 言語プログラム) の集まりです。

Toolbox は、制御、信号処理、システム同定やその他の分野の第一線の研究者の方々の努力の成果です。

インストールの内容によっては、すべての Toolbox を含んでいない可能性があります。インストールにどの Toolbox が含まれているかを確認するには、左ペインの Toolboxes エントリを展開します。そこから、デモを表示するために Toolbox を選択し、次に表示して実行する個々のデモを選択してください。[ヘルプ] 内の例も見ることができます。ドキュメンテーションからサンプル コードを表示、実行、またはコピーするには、[コンテンツ] ペインで、製品を選択し、[例] を選択してください。

デモのある製品のリストを参照するには、www.mathworks.com/products/demos をご参照ください。
