Simulink® Power Window Controller and Detailed Plant Model

A more detailed plant model that includes power effects in the electrical and mechanical domains is used to validate that the force exerted by the window on a trapped object never exceeds 100 [N].

You must install SimMechanics™ and SimPowerSystems™ to run this power window demo.
