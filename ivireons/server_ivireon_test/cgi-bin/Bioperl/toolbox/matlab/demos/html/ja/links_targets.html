リンクとターゲットについて

リンクとターゲットは、組み込みシステムの開発のための特別な機能で MATLAB と Simulink を拡張します。

  • リンクは、特定の統合開発環境 (IDE)、またはハードウェア (HDL) シミュレータのための組み込みコードを生成、ビルド、検証、デバッグ、展開します。
  • ターゲットは、デバイス ドライバーの統合や、リアルタイムのオペレーティング システム (RTOS)、プロセッサーとプログラミング可能なハードウェアに対するアセンブリレベルのコードの最適化のような、組み込みのプラットフォームをサポートします。

個々のインストールは、すべてのリンクとターゲットを含んでいない可能性があります。別のリンクとターゲットは、別の機能を持っています。インストールに含まれているリンクとターゲットを参照し、それらの異なる機能を確認するには、左のペインのリンクとターゲットの項を展開してください。そこから、リンクまたはターゲットを選択してそのデモを表示し、次に表示して実行する個々のデモを選択してください。[ヘルプ] 内の例も見ることができます。ドキュメンテーションからサンプル コードを表示、実行、またはコピーするには、[コンテンツ] ペインで、リンクまたはターゲットを選択し、[例題] を選択してください。
