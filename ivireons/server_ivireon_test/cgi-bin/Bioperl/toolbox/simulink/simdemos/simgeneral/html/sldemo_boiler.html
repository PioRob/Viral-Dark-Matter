Bang-Bang Control Using Temporal Logic

This demonstration shows how to use Stateflow® to model a bang-bang temperature control system for a boiler. The boiler dynamics are modeled in Simulink®.

The Bang-Bang Controller Stateflow chart shows how to use functions and Subcharts within Stateflow.
