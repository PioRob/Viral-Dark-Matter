Custom Regressors Import Dialog

This dialog facilitates import of custom regressor objects from MATLAB workspace or from a MAT file into the GUI’s Model Regressors window. Choose the corresponding radio button to select either the Workspace or a MAT file as the source to import the regressor objects from. For a MAT file, you may either specify the name of the file explicitly (with qualified path, if file is not located in the present working directory or exists on MATLAB path), or you can browse for it by clicking on the Browse button.

Custom regressors may be created explicitly using customreg function. A set of polynomial type custom regressors may also be created using the polyreg function.

Example: Creating two custom regressors as an object array:

    cr1 = @(x,y) x*sin(y); cr2 = @(x) x^3;
    C = [customreg(cr1,{'u1' 'y1'},[1 3]),customreg(cr2,{'u1'},2)]

Example: creating polynomial type regressors for a model with orders: na=2, nb=2 and nk=1:

   M = idnlarx([2 2 1], 'linear'), 
   C = polyreg(M)
