Multi Output Graphical Functions

This model demonstrates a new feature in Stateflow® R2008a, namely, the ability to have multiple outputs from a graphical function. The syntax for calling a function with multiple outputs is exactly the same as in MATLAB®. Please see the section Using Graphical Functions to Extend Actions for more information.
