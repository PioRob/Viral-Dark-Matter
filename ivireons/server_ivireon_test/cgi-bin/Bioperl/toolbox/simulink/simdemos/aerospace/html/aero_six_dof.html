Six Degrees of Freedom (6-DoF) Motion Platform

This demonstration shows how to model six degrees of freedom motion in Simulink®. The Block choice item on the Configurable Subsystem's right-click menu provides options for switching between using Euler Angles and Quaternions to model the equations of motion.
