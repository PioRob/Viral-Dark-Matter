Advanced Properties of Wavelet Network

This dialog allows configuration of advanced properties of Wavelet Network.

FinestCell: Integer or string giving the minimum number of data points in the smallest cell (a cell is the area covered by the significantly non zero part of a wavelet). Default value: 'auto', meaning that it is determined from data.

MinCells: Integer (default 16). Minimum number of cells in the partition.

MaxCells: Integer (default 128). Maximum number of cells in the partition.

MaxLevels: Integer (default 10). Maximum number of wavelet levels.

DilationStep: Real scalar (default 2). Dilation step size.

TranslationStep: Real scalar (default 1). Translation step size.

Type "idprops wavenet" in MATLAB Command Window for more information on Wavelet Networks.
