Foucault Pendulum Model with VRML Visualization

This model solves the differential equations for the Foucault Pendulum problem and displays the pendulum bob movement in the VRML scene. You can modify the Pendulum location by changing the Latitude / Longitude constant values in the model and other parameters (g, Omega, L and initial conditions) in MATLAB® workspace.

There are two VRML scenes associated with this model. The first scene shows the Foucault pendulum itself, the second shows geographical position of the pendulum on the Earth. During the simulation, two arrow markers indicate the current and the initial position of the pendulum in the Earth coordinate system.
