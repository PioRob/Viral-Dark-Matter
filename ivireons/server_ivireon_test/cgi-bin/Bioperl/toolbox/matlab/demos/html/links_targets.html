About Links and Targets

Links and Targets extend MATLAB and Simulink with specialized functionality for embedded systems development.

  • Links generate, build, verify, debug, and deploy embeddable code for specific integrated development environments (IDEs) or hardware (HDL) simulators.
  • Targets provide embedded platform support such as integration of device drivers, real-time operating systems (RTOS), and assembly-level code optimizations for processors and programmable hardware.

Your particular installation might not include all links and targets. Different links and targets have different capabilities. To see which links and targets your installation includes and to review their distinctive capabilities, expand the Links and Targets entry in the left pane. From there, select a link or target to view its demos, and then select individual demos you want to view and run. You can also find examples in Help. In the Contents pane, select a link or target and choose Examples to view, run, or copy sample code from the documentation.
