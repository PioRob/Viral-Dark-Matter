Confidence Intervals

Confidence intervals are toggled on/off in the Options menu.

Any estimated model has a degree of uncertainty, which affects the reliability of the various model properties. Therefore, all model views have estimated accuracy measures (except the K-step ahead prediction in the Model output view).

The uncertainty region is marked by two dash-dotted lines, one on either side of the nominal model curve with the same color as the curve.

The statistical interpretation is that (with the indicated probability) the true system response is found within the marked confidence region.

Caution: This interpretation assumes that the system can be described within the chosen model class and that the model has passed validation tests. In addition, the calculations assume that the model errors are small.

Yuo can set the confidence level as the number of standard deviations of a Gaussian distribution in a dialog, opened by selecting Options->Set confidence.

Note: In the Model resids and the Transient resp (Impulse Response and Stem Plot) views, the confidence lines are on either side zero.

See BODE, NYQUIST, TFDATA, SSDATA and ZPKDATA in the manual for more information about uncertainty measures.
