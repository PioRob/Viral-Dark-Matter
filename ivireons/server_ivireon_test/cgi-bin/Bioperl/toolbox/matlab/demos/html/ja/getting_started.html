デモのご利用の前に

デモの選択

左ペインの製品エリアを展開してデモのリストを表示してください。右ペインにサムネイル イメージと各デモのタイプが表示されます。次のタイプのデモがあります。
 M ファイル
ソース コード、解説、出力を含む段階的にストーリーを追って説明するデモです。それらのデモは、エディター/デバッガーのセルとファイル変換機能を使ってM ファイル スクリプトから HTML に変換されています。
M GUI
機能を調べるスタンドアロン ツールです。
モデル
ブロック線図です。
ビデオ
ツールの重要な機能を紹介するムービーです。これらは Flash Player が必要です。いくつかはインターネット接続も必要です。

デモの実行

リストからデモを選択し、右ペインでデモについての情報を表示します。上部にあるリンクから、以下のアクションを実行します。
開く...
M ファイルと M GUI タイプのデモについて、エディター/デバッガーでソース コードを表示します。[セル] > [現在のセルを実行して進める] メニュー項目を使用して、一度に 1 セクション (セル) ずつ M ファイル タイプのデモを実行することができます。

モデル タイプのデモについて、Simulink のブロック線図を開きます。
実行...
ツールの重要な機能を紹介するムービーです。これらは Flash Player が必要です。いくつかはインターネット接続も必要です。

等価な機能を持つ関数

ヘルプ ブラウザーでデモを開くには、コマンド ウィンドウで demo を実行してください。特定の製品、またはカテゴリのデモ ページへ直接移動するには、引数を追加してください。

たとえば、次のように実行して MATLAB グラフィックスのデモをリストします。 
demo matlab graphics
詳細は、関数 demo のリファレンス ページを参照してください。
