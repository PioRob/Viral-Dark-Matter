デモが見つかりません

MATLAB は以下のいずれかの理由により、指定したデモを見つけることができませんでした。

  • 引数を持つ関数 demo を実行しましたが、MATLAB はその引数と一致するデモのサブトピックまたはカテゴリを見つけることができませんでした。
    別の引数で demo を実行してみてください。詳細は、関数 demo のリファレンス ページを参照してください。

  • ヘルプ ブラウザーの設定により、指定した製品のデモへのアクセスが制限されています。
    インストールされているすべての製品のデモにアクセスできるように以下の設定を行います。
    1. [ファイル] > [設定] を選択し、リストから [ヘルプ] を選択します。
    2. [ヘルプ] ペインの [製品毎のフィルター] で、[すべての製品] を選択してください。
    3. [設定] ダイアログ ボックスの [OK] をクリックします。
    4. インストールされているいずれかの製品のデモにアクセスしてみてください。
