Using Hot-Keys in the Ident Window:

You can speed up certain actions by pressing a key when the ident window is in focus. These following hot-keys are available:

c creates a correlation model
s creates a spectral model
q performs Estimate->Quick start
d opens the Import Data dialog
m opens the Import Model Object dialog

Working with data and model icons in the ident window:

Click a data or model icon to select or clear the selection. See Views Windows to learn how to create views of data and models.
Icons can by dragged-and-dropped by holding the mouse button down.
Double-click an icon to open the Data/Model Info dialog.

Time Savers
Click and hold down the left mouse button to drag and drop.
Use the middle button (extend select) on a three-button mouse (same as Shift+click) to toggle a selection.
Right button (Alt select), Ctrl+click and double-click any button all open the Info dialog.

Mouse behavior in the view windows (such as Time Plot or Model Output):

When Style->Zoom is enabled: Left click on the plot zooms in, right click zooms out, double click restores default magnification.
When Style->Zoom is enabled: Click with middle button (or Shift-click) on a plot to see the name of a curve and the data value at that location.
When Style->Zoom is disabled: Click on plot to see the name of a cure and the data value at that location.
