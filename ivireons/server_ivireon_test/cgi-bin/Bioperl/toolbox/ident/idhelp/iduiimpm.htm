Importing Models

This dialog is used to import models from the MATLAB workspace. The imported model is added to the Model Board in the ident window.

Model Variable
Enter a model variable name in the editable field. The variable must exist in the workspace and must have a valid model format. The following formats are accepted:

LTI
Model defined as a linear time invariant (LTI) system in the Control System Toolbox™: SS, TF, ZPK, or FRD.
IDMODEL
IDMODEL object in the System Identification Toolbox™: IDPOLY, IDSS, IDARX, IDGREY, or IDPROC.
IDFRD
Frequency domain model as identified frequency response data.
See IDHELP and IDPROPS reference for details.

Optional Model Information
Notes
Enter any text that describes the model. The notes remain associated with the model.

Click Import to import the model.

To restore default field values in this dialog, click Reset.
