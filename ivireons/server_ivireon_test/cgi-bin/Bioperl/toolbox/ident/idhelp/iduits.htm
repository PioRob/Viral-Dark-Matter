A time series is a measured "output" signal that does not have a known, measured input. A multivariate time series is a vector-valued signal.

The analysis problem for time series and signals might be to investigate their frequency contents (spectra), and to build predictors for them. A parametric model of a time series describes how it could be generated from a white noise source.

When you import scalar and multivariate time series data by using the Import Data dialog, leave the Input field empty.

To generate time series spectra, select the Noise spectrum check box under Model Views in the ident window.

To compare the measured and predicted time series values, select the Model output check box under Model Views.
