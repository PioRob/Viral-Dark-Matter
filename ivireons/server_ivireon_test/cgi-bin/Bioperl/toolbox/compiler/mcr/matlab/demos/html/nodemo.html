Demo Not Found

MATLAB could not find the demo you specified for one of these reasons:

  • You ran the demo function with an argument, and MATLAB could not find a demo subtopic or category matching that argument.
    Try running demo with a different argument. For more information, see the reference page for the demo function.

  • A Help browser preference is restricting access to demos for the product you specified.
    To set the preference so demos for all installed products are accessible:
    1. Select File > Preferences, and then select the Help entry from the list.
    2. In the resulting Help pane, for Filter by Product, select All Products.
    3. Click OK in the Preferences dialog box.
    4. Try accessing demos for any installed product.
