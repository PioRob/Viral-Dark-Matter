Importing Data

You can load a saved session by selecting its name from the File menu.

To import a single data set directly from the MATLAB workspace, do one of the following:

  1. To create a data set from input and output signals available in the workspace, select
    Import data->Time domain data
    or Import data->Frequency domain data
    from the pop-up menu in the ident window.
  2. If the data set already exists in the workspace as an IDDATA or IDFRD object, then import it by selecting
    Import data->Data Object
Note: Use the hot-key d to open the Import Data dialog when the ident window is in focus.

In the Import Data dialog box, enter the appropriate information, such as the Starting time and the Sampling interval. Click the More button to display additional fields in the dialog, such as for entering Notes.

Importing Models

Importing Models

You can load a saved session by selecting its name from the File menu.

To import saved models in the IDMODEL and IDFRD formats from the MATLAB workspace, use the Import Models pop-up menu. You can also import models in the Control System Toolbox LTI model formats.

Ident Sessions

Ident Sessions

A saved session stores the status of the data sets and models that are currently in the Model and Data Board(s) - including their diary and notes. The session also includes an arbitrary number of additional boards (opened by selecting Options->Extra model/data board in the ident window).

You open, manage, save and close sessions by selecting commands from the File menu in the ident window.

Only one session can be open at any time. You can, however, merge a previously-saved session into the current one by selecting File->Merge session from the menu.
