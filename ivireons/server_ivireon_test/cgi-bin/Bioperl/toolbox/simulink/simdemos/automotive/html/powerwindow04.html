Simulink® Power Window Controller With Data Acquisition Effects

In moving to an implementation, additional effects that may affect the control have to be accounted for. This model includes phenomena such as signal conditioning to measure the armature current and quantization of the measurement.

You must install SimMechanics™, SimPowerSystems™, Signal Processing Blockset™, and Simulink® Fixed Point™ to run this power window demo.
